## 项目说明

基于mvc和模板技术（演示模板如何工作），这只是一个适合初学php的开发者学习的demo项目，简单的实现了mvc的实现原理以及怎么使用smarty模板技术，因此并不具备严格意义上的安全性和稳定性，不适用于生产环境。

**更新一次**

2015年11月5日 02:17:33 更新项目view层 使用smarty
